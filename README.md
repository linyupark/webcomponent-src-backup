# 通用 Web 组件库

### 开发组件

`npm start`

### 打包组件

针对客户配置的需要修改 stencil.config.ts

`npm run build` -- 全量打包 -> dist/

`npm run build-客户名` -- 针对性打包 -> dist/客户名
